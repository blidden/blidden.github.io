Accelo - Create Screens [iOS & Android]

Create screens for the Accelo iOS and Android apps - updated to align with the new create flows of the web app.

Background

The create screens in Accelo serve as the genesis of creating modules. Each module has a unique set of field elements (both mandatory and optional) which help the user shape the object to their need.

While not all module create flows are available on the app due to back-end limitations, several small to medium based modules can be created on the fly via the Accelo iOS and Android offerings.

The Android app was influenced by the material design framework, but with enough visual nudges to ensure the Accelo branding remained a constant throughout. Helpful UX workflows, such as pre-filling Contact details based on local phone storage gave way for a seamless create experience.

The iOS app followed many of the flows from the Android offering, adding in geo-fencing capabilities when logging time on the Create Timer flow.
